I've been pretty lenient with my daily scheduling so far, but once I've got a chunk of work lined up for October, I'm going to need to be more disciplined. In order to keep track of tasks, I created an account for the task management software that Formula Slug used for FSAE: Trello, and added several of the tasks that I've outlined in this blog. I feel really good about setting all this up, and it works more as a true application of all the techniques I've learned through intra and extracurricular studies. I'm excited to make some solid progess.
